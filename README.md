# Redmine Favorites Plugin

Плагин для Redmine, который позволяет пользователям отмечать задачи как избранные и быстро получать к ним доступ.

## Функциональность

* Добавление задач в избранное
* Удаление задач из избранного
* Просмотр списка избранных задач
* Иконки избранного на странице задачи, в списке задач и на странице редактирования
* Пункт в контекстном меню для добавления/удаления задач из избранного
* Пункт в глобальном меню для быстрого доступа к избранным задачам
* Поддержка стандартных фильтров Redmine для списка избранных задач

## Установка

1. Клонируйте репозиторий в директорию plugins вашего Redmine:

```bash
cd /path/to/redmine/plugins
git clone https://github.com/yourusername/redmine_favorites.git
```

2. Установите зависимости (если есть):

```bash
cd /path/to/redmine
bundle install
```

3. Выполните миграции базы данных:

```bash
cd /path/to/redmine
rake redmine:plugins:migrate RAILS_ENV=production
```

4. Перезапустите Redmine.

## Использование

* Для добавления задачи в избранное, нажмите на иконку звездочки на странице задачи или в списке задач
* Для просмотра избранных задач, нажмите на пункт "Избранное" в глобальном меню
* Для удаления задачи из избранного, нажмите на иконку звездочки еще раз
* Для массового добавления/удаления задач из избранного, выберите нужные задачи и используйте соответствующий пункт в контекстном меню

## Особенности

* AJAX-обработка добавления/удаления задач из избранного без перезагрузки страницы
* Анимация при добавлении/удалении задач из избранного
* Автоматическое скрытие уведомлений
* Поддержка SVG-иконок для лучшего качества отображения
* Визуальная индикация процесса добавления/удаления задач из избранного
