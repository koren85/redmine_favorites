# Плагин Избранного для Redmine

Плагин для Redmine, который позволяет пользователям отмечать задачи как избранные и быстро получать к ним доступ.

[English version](README.md)

## Возможности

* Добавление задач в избранное
* Удаление задач из избранного
* Просмотр списка избранных задач
* Иконки избранного на странице задачи, в списке задач и на странице редактирования
* Пункт в контекстном меню для добавления/удаления задач из избранного
* Пункт в глобальном меню для быстрого доступа к избранным задачам
* Поддержка стандартных фильтров Redmine для списка избранных задач
* Фильтр избранного в стандартных запросах Redmine ("Мои избранные")
* Колонка с иконкой избранного в списке задач (перед номером задачи)
* Виджет для личной страницы с избранными задачами

## Установка

1. Клонируйте репозиторий в директорию plugins вашего Redmine:

```bash
cd /path/to/redmine/plugins
git clone https://github.com/yourusername/redmine_favorites.git
```

2. Установите зависимости (если есть):

```bash
cd /path/to/redmine
bundle install
```

3. Выполните миграции базы данных:

```bash
cd /path/to/redmine
rake redmine:plugins:migrate RAILS_ENV=production
```

4. Перезапустите Redmine:

```bash
touch tmp/restart.txt
```

## Использование

### Базовые операции

* Для добавления задачи в избранное нажмите на иконку звездочки на странице задачи или в списке задач
* Для просмотра избранных задач нажмите на пункт "Избранное" в верхнем меню
* Для удаления задачи из избранного нажмите на иконку звездочки еще раз
* Для массового добавления/удаления задач из избранного выберите нужные задачи и используйте соответствующий пункт в контекстном меню

### Расширенные возможности

#### Фильтр избранного в запросах

Вы можете фильтровать задачи по статусу избранного в любом запросе Redmine:
1. Перейдите в любой список задач
2. Нажмите "Добавить фильтр"
3. Выберите "Избранные задачи"
4. Выберите "Да" для отображения только избранных или "Нет" для исключения избранных

#### Виджет для личной страницы

Добавьте виджет избранных задач на вашу личную страницу:
1. Перейдите в "Моя страница" (верхнее правое меню → "Моя страница")
2. Нажмите "Настроить страницу"
3. Найдите "Избранные задачи" в доступных блоках
4. Перетащите его в нужное место
5. Нажмите "Сохранить"

Виджет отображает 10 последних избранных задач.

## Особенности

* AJAX-обработка добавления/удаления задач из избранного без перезагрузки страницы
* Плавная анимация при добавлении/удалении задач из избранного
* Автоматическое скрытие уведомлений
* Поддержка пользовательских иконок для лучшего качества отображения
* Визуальная индикация процесса добавления/удаления задач из избранного
* Персональный список избранного для каждого пользователя (защита приватности)
* Совместимость с плагином additional_tags и другими популярными плагинами Redmine
* Глобальный плагин - не требует включения для каждого проекта
